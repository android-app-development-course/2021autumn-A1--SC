# 2021autumn-A1--校园集市
#### 产品名称：
校园集市

#### 产品定位：
针对大学生闲置交易、失物招领的信息平台

#### 目标用户：
大学生

#### 产品Slogan：
共享闲置，共寻失物

#### 产品介绍：
这是一款针对在校大学生的闲置物品交易平台和失物信息发布平台，以专注集中的信息服务，为大学生的生活创造便利。
